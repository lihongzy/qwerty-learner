# Qwerty Learner 项目描述

## 简历一句话版

Qwerty Learner 是一个将单词记忆、拼写巩固与英文键盘输入训练结合的练习系统，支持章节化词库训练、错题复习与本地数据统计。

## 简历精简版

- 独立设计并实现英文输入训练项目 Qwerty Learner，将词汇学习、拼写纠错与键盘输入训练整合为同一套练习流程。
- 构建章节练习、结果统计、错题沉淀、复习模式和本地持久化能力，形成可量化、可持续的日常训练闭环。

## 简历技术版

- 基于 React、TypeScript、Jotai、Dexie、Radix UI 与 Tauri 构建英文输入训练系统，完成 Web 与桌面端的统一实现。
- 设计章节练习、错词记录、复习模式、结果统计和偏好配置等核心状态流，并通过本地数据库沉淀训练数据。
- 围绕速度、正确率、错词分布和章节成绩建立练习反馈机制，将输入行为转化为后续复习依据。

## 简历亮点版

- 将“记住单词”和“打对单词”整合为同一训练过程，而不是传统的分离式背词或打字练习。
- 支持章节化练习、即时反馈、错题本和复习模式，形成完整训练闭环。
- 强调真实英文输入场景下的拼写准确率、输入速度和肌肉记忆训练。
- 采用本地优先的数据存储方案，降低使用门槛并提升持续使用体验。

## 面试表达版

这个项目的核心不是做一个词库展示工具，而是把单词记忆、拼写巩固和键盘输入训练融合在一起。用户在练习过程中不仅要认识单词，还要能准确、快速地输入出来。项目通过章节化词库、即时输入反馈、错词沉淀、复习模式和本地训练记录，形成了一套低摩擦、可量化、可长期坚持的训练系统。
